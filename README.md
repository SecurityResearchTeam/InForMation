# InForMation
windows主机信息收集工具(服务器信息，系统日志，中间件日志等)

\>information.exe -i start -L start -s start  运行所有功能

\>information.exe -i start  运行收集系统信息功能

1.运行程序，会在当前目录下生成采集的日志以及系统信息文件夹。

2.程序运行期间，可以输入目标磁盘盘符，对该盘进行扫描，获取.log文件；如果不输入直接回车，默认为全盘扫描。
